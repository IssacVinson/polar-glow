# Polar Glow Detailing

Flutter mobile app for car detailing services in Alaska (Anchorage area).

## Features
- Customer: Browse services → Book appointment → Choose detailer or next available
- Employee: Clock in/out, set availability (bulk + per-day), regions (Anc, Wasilla, ER, Base)
- Admin: Promote users, track employees, view schedules/hours
- Firebase: Auth (email only), Firestore for data

## Tech Stack
- Flutter (Dart)
- Firebase (Auth + Firestore)
- Packages: provider, intl, table_calendar

## Setup
1. `flutter pub get`
2. `flutter run -d chrome` (web)
3. Or connect Android/iOS device/emulator

## Firebase
- Config in `firebase_options.dart` (generated by `flutterfire configure`)
- Security rules allow user-owned data + admin read access

## Folders
- `lib/screens/`: All UI screens (auth_wrapper.dart, booking_screen.dart, etc.)
- `lib/providers/`: AuthProvider
- `assets/images/`: logo.png, google_logo.png (if used later)
